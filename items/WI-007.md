# 执行 MVP 功能、异常与权限验收

## 任务说明

基于集成应用 Artifact 和 PRD V1.7，建立可重复执行的测试方案并执行关键用户主线、管理主线、异常状态、移动端体验和权限隔离验收。

## 必需交付

- 可追溯到 PRD 验收条目的测试用例或自动化测试；
- 测试环境、数据、步骤和命令说明；
- 注册登录、聊天、历史会话、管理端和 LLM 调用记录的结果；
- 停止、失败、重试、弱网或重复点击等异常场景结果；
- 会话越权、管理端访问控制和凭据泄漏检查结果；
- 缺陷清单、复现步骤、严重级别和复测结果；
- 一份明确的通过或不通过结论。

## 验收标准

- 覆盖 PRD 第 11 节及 Mission 验收标准中的关键主线；
- 验证普通用户无法访问其他用户会话或管理端；
- 验证每次聊天、标题和重试调用记录与实际行为一致；
- 验证 H5 目标宽度、软键盘、安全区、侧边栏和滚动行为；
- 所有阻断 Mission 验收的缺陷均已关闭并复测通过；
- 结果可由其他一级 Agent 或 Mission Owner 重复检查。
