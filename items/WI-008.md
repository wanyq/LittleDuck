# 准备腾讯云部署配置与运维文档

## 任务说明

基于集成应用 Artifact，为单台腾讯云服务器形成可重复执行的生产部署、更新和回滚方案。不得把任何真实服务器凭据或 OpenAI API Key 写入仓库。

## 必需交付

- 生产构建与运行配置；
- 环境变量模板和安全注入说明；
- 数据库初始化、迁移和备份说明；
- 服务启动、自启动、健康检查、日志查看和进程重启方式；
- HTTPS、反向代理或公网访问所需配置说明；
- 版本更新、数据保护和回滚步骤；
- 一份从干净服务器开始的部署检查清单。

## 验收标准

- 可在目标腾讯云 Linux 服务器重复执行；
- 用户端、管理端和服务端的公网路径及端口关系明确；
- 生产凭据不进入 Git 历史、日志示例或配置模板；
- 服务器重启后的服务恢复方式明确；
- 更新和回滚步骤具备可操作性并说明数据风险；
- 健康检查与日志足以支持上线验证和故障定位。
